## Phase I experiment package: weighted-LSE DP

This directory hosts the Phase I experimental program for the weighted-LSE
dynamic-programming / reinforcement-learning project (classical β=0 baselines,
dynamic-programming ground truth, and the first wave of RL ablations). The
authoritative implementation contract is `docs/specs/phase_I_classical_beta0_experiments.md`;
that spec — not this README — governs the expected modules, configs, metrics,
and result layout. Later phases (II–III) will extend this scaffold with
safety / paired-run tooling under the same top-level structure.
