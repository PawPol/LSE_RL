# Recomputed Phase I-IV tables (WP0 audit)

This directory is regenerated by
`scripts/audit/run_consistency_audit.py`. Each CSV here is a
recomputation of a committed table under
`results/weighted_lse_dp/processed/` or `results/processed/`, derived
directly from `calibration_stats.npz` / aggregated `summary.json`
files.

Files are namespaced by phase:

    recomputed_tables/
      phase3/clip_activity.csv      recomputed version of the
                                     `tab:clip-activity` contents from
                                     the paper; one row per Phase III
                                     task, columns = clip_fraction_mean,
                                     effective_discount_mean,
                                     beta_used_abs_mean.
      phase4B/P4B_A.csv             recomputed version of
                                     `results/processed/phase4b/P4B_A.csv`;
                                     per-(task, algorithm) mean_return.

Do not edit by hand; the audit script overwrites these files. Diffs
against the committed reference tables are emitted into
`results/audit/consistency_report.json` under
`check == "table_diff"` — BLOCKER rows indicate the recomputed values
disagree with the committed tables beyond tolerance.
