## Function inventory (spec §7.2)
1. `contraction_reward(R, eps=1e-8) -> ndarray` — `M_e = log(R_e+ε) - log(R_{e+1}+ε)`.
2. `empirical_contraction_ratio(R, eps=1e-8) -> ndarray` — `(R_{e+1}+ε)/(R_e+ε)`.
3. `log_residual_reduction(R, eps=1e-8) -> ndarray` — alias of (1).
4. `ucb_arm_count(arms, n_arms=7) -> ndarray[int64]` — per-arm pull counts; raises on out-of-range.
5. `ucb_arm_value(rewards, arms, n_arms=7) -> ndarray[float64]` — per-arm Welford running mean; 0.0 if never pulled.
6. `beta_clip_count(raw, used, atol=1e-8) -> int` — `|raw - used| > atol`.
7. `beta_clip_frequency(raw, used, atol=1e-8) -> float` — count / len; 0.0 on empty.
8. `recovery_time_after_shift(ret_smooth, shift_ep, threshold) -> int` — `e - shift_ep`; -1 if never reached or shift_ep out of range.
9. `beta_sign_correct(used, oracle) -> ndarray[bool]` — `np.sign(used) == np.sign(oracle)` (oracle==0 ⇒ used==0 only correct).
10. `beta_lag_to_oracle(used, oracle) -> ndarray[int64]` — `e - argmin_{e'≤e} |used[e'] - oracle[e]|`; ties resolve to oldest match.
11. `regret_vs_oracle(oracle_ret, method_ret) -> float` — `sum(oracle - method)`; raises on length mismatch.
12. `catastrophic_episodes(returns, theta_low) -> int` — `sum(returns <= theta_low)`.
13. `worst_window_return_percentile(returns, window=100, pct=5.0) -> float` — percentile of rolling-window means; falls back to single window if `len < window`.
14. `trap_entries(state_traj, trap_states) -> int` — count of new entries into trap region.
15. `constraint_violations(log) -> int` — `np.count_nonzero`.
16. `overflow_count(q_abs_max, threshold=1e6) -> int` — `sum(q >= threshold)`.

Module constant `EPSILON = 1e-8` (mirrors `tab_operator._EPS_BETA`).
