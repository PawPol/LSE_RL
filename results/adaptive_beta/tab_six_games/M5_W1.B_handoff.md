## Implementation notes

- **AUC computation** uses `numpy.trapezoid` over the per-(game, subcase,
  method, seed) `return` curve, then averages seed-level AUCs and reports
  seed-bootstrap SE (`std(ddof=1) / sqrt(n)`) per spec §11 conventions.
- **Switch-aligned plots** stack ±50-episode windows around every
  `switch_event == 1` row, grouped by `(run_id, seed)`. Empty slices and
  `n < 2` cases are guarded so no `nanmean / nanstd` RuntimeWarnings are
  emitted on sparse fixtures.
- **Contraction arm-prob panel** reports the seed-fraction selecting each
  arm at each episode under method `contraction_ucb`, smoothed via a
  50-episode rolling mean with edge-padding (no future leakage).
- **Safety panel aggregations** are grouped bar charts: catastrophic
  count = `sum`, beta-clip frequency = `mean`, worst-window return = p5
  via `np.percentile(..., 5)`.
- All figures use `matplotlib` with `Agg` backend; no seaborn, no
  pandas-plot magic.
